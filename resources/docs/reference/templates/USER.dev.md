## 建议记录的内容

- 对方主要负责的项目、仓库或系统
- 对方对修改、验证、提问频率和交付方式的偏好
- 对方最关注的是速度、稳定性、可维护性，还是合规性
- 哪些操作可以直接做，哪些操作必须先确认
- 是否有固定的输出格式、分支命名或报告要求

## 说明

这个文件用于帮助你更准确地协作，不是为了收集无关信息。只记录对任务执行真正有帮助、且适合长期保留的内容。
