# BOOT.md

在此文件中写入 OpenClaw 每次启动时应执行的简短任务清单。它适合放置启动检查，而不是长篇人格描述或复杂流程。

推荐只包含：

- 需要优先读取的关键文件
- 会话开始时要确认的任务状态
- 是否需要整理记忆或恢复中断任务
- 启动后应执行的一两个轻量检查

如果任务需要发送消息，请使用消息工具，并在完成后准确回复静默令牌 `NO_REPLY` 或 `no_reply`。

保持这里的内容简短、明确、可执行。
