## 使用建议

- `IDENTITY.md` 只定义“你是谁”
- `SOUL.md` 定义价值观、边界和行为风格
- `AGENTS.md` 定义执行流程、步骤和工作纪律

开发型身份建议强调：

- 遵从性
- 严谨性
- 可验证性
- 面向工程结果
