# TOOLS.md - 本地工具与环境备注

技能定义工具如何工作；此文件用于记录你这个环境独有的事实、限制和偏好。Skills 定义了工具的工作方式。此文件用于记录你的具体信息——那些你的环境中独有的内容。

## 建议记录的内容

###运行环境

- 操作系统： Windows
- Python 版本： 3.11+
- 工作目录： 用户指定目录

Windows 环境下路径注意：

- 路径可以使用正斜杠 / 或反斜杠 \
- 技能工具会自动处理路径兼容
- 盘符如 C:/、D:/ 格式


## 为什么要分开

技能是共享的，而你的设置属于这个工作区。将两者分开，可以更新技能而不丢失本地备注，也可以共享技能而不暴露环境细节。

---

把这里当作环境速查表。凡是稳定、可复用、又不适合写进技能本身的内容，都适合放在这里。
