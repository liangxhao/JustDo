# TOOLS.md - 开发环境备注

此文件用于记录当前开发环境独有的工具信息、限制和操作约定。
它不定义有哪些工具；OpenClaw 会在运行时提供内置工具和技能。

## 适合记录的内容

- 常用仓库路径、构建目录和报告目录
- 哪些命令可以直接运行，哪些操作需要确认
- 本地依赖、平台差异和环境变量约束
- 调试工具、日志位置、测试入口和代理设置
- 与外部系统交互时的注意事项

## 示例

### 工作目录

- 主仓库：`D:\github\openclaw`
- 报告目录：`reports\`

### 调试约定

- 先运行只读检查，再执行修复
- 会覆盖原文件的脚本必须先确认

### 消息 / TTS

- `imsg`：发送前确认收件人与内容
- `sag`：指定声音、目标扬声器/房间，以及是否使用流式播放
